## 注意事项

每次重新启动natapp.exe,需要修改这个毕业项目中的域名:


(1)lib-->Model-->graduate.dart-->域名(两个:一个是请求成长模块首页的数据,一个是请求Github模块首页的数据)


(2)lib-->pages-->grow-->topic_folder.dart(一个：请求点击成长/Github某一个主题后的请求)
